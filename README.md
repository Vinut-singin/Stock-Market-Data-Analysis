# 📈 Stock Market Data Analysis

A comprehensive Exploratory Data Analysis (EDA) project on historical stock market data of **MARUTI**, **ULTRACEMCO**, and **NESTLEIND** using Python. The project analyzes long-term trends, stock volatility, price fluctuations, and investment opportunities using financial performance metrics.

---

## 📌 Project Objectives

- Analyze long-term market trends using yearly average closing prices.
- Study trading volume and price fluctuations.
- Compare the performance of multiple stocks.
- Evaluate investment opportunities using financial metrics such as CAGR, Volatility, and Maximum Drawdown.

---

## 📊 Dataset

The dataset contains historical stock market information for:

- MARUTI
- ULTRACEMCO
- NESTLEIND

Each dataset contains the following attributes:

| Column | Description |
|---------|-------------|
| Date | Trading Date |
| Symbol | Stock Symbol |
| Open | Opening Price |
| High | Highest Price |
| Low | Lowest Price |
| Close | Closing Price |
| Volume | Trading Volume |

---

## 📈 Analysis Performed

### 1. Market Trend Analysis
- Calculated yearly average closing prices.
- Compared long-term growth trends of all three companies.

---

### 2. Volume and Price Fluctuation
- Identified significant price gap days.
- Analyzed trading volume alongside price movements.

---

### 3. Comparative Analysis
- Compared percentage change in stock prices.
- Evaluated yearly performance across companies.

---

### 4. Stock Volatility
- Calculated the standard deviation of daily closing prices.
- Compared stock risk levels.

---

### 5. Investment Recommendation
Calculated and compared:

- Compound Annual Growth Rate (CAGR)
- Maximum Drawdown
- Volatility

A weighted scoring approach was used to recommend the stock offering the best balance between return and risk.
